# Task 2: Hidden Item Game

This repository contains the solution for **Task 2: Hidden Item** of the Fullstack Engineer Assessment Test for PT FOMO INOVASI TEKNOLOGI.

## Description
The objective of this task is to build a simple command-line program for a hidden item game. The script calculates all probable coordinates where a hidden item might be located based on specific movement rules within a grid. 

## Tech Stack
- **Language:** PHP (CLI)
- **Approach:** Object-Oriented Programming (OOP)

## How to Run
Since this is a standalone PHP script, no external frameworks or dependencies are required. You can run the program directly from your terminal using the PHP CLI.

1. Navigate to the directory containing the script.
2. Execute the following command:
   ```bash
   php index.php
