# FOMO INOVASI TEKNOLOGI - Fullstack Engineer Assessment

This repository contains the completed technical assessment for the Fullstack Engineer position at FOMO INOVASI TEKNOLOGI.

## Repository Structure

The assessment consists of two separate tasks, each isolated in its own directory. Please navigate to each folder for detailed setup instructions, API endpoints, and architectural documentation.

### 📁 [Task 1: FomoStore API](./task-1)
A RESTful API built with Laravel 13, focusing on an online product ordering system with robust flash sale stock protection.
- **Key Highlights:** Clean architecture, service pattern, database transactions, and race condition prevention using row locking (`lockForUpdate`).
- **Tech Stack:** Laravel 13, PHP 8.4, PostgreSQL, Docker Compose.
- 📖 [View Task 1 Documentation](./task-1/README.md)

### 📁 [Task 2: Hidden Item Game](./task-2)
A command-line interface (CLI) application built with pure PHP that calculates all probable coordinates for a hidden item on a defined grid based on specific movement rules.
- **Key Highlights:** Object-Oriented Programming (OOP), algorithmic grid parsing, path validation, and visual grid output mapping.
- **Tech Stack:** PHP (CLI).
- 📖 [View Task 2 Documentation](./task-2/README.md)
